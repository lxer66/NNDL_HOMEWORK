## 简要介绍

本次作业的目标是实现生成模型，具体包括以下内容：

### 1. GAN 实现
- 使用普通 GAN 模型在 CIFAR-10 数据集上训练生成器。
- 提供训练脚本 `train.py` 和推理脚本 `inference.py`。
- 训练过程中保存模型权重和生成图片，实验结果记录在 `实验报告.md` 中。

### 2. cGAN 实现（附加题1）

- 在 GAN 的基础上实现条件生成网络 (Conditional GAN)，支持生成特定类别的图像。
- 提供训练脚本 `train.py` 和推理脚本 `inference.py`。
- 训练过程中保存模型权重和生成图片，实验结果记录在 `实验报告.md` 中。

### 3. 实验报告
- 每个模型的文件夹中包含 `实验报告.md`，详细记录了模型原理、实验设置、实验结果（如损失曲线、生成图片示例）等内容。

### 4. 调研报告（附加题2）
- 文件 `调研报告.md` 提出了一个创新的图像生成问题：**基于深浅层分离的零样本角色一致性生成研究**。
- 报告详细分析了当前生成模型在角色一致性生成中的挑战，并提出了一种名为 **Layer-Adaptive Feature Injection (LAFI)** （层级自适应特征注入）的创新解决方案。
- 该方法通过分层级的注意力路由机制，解决了现有方法中的“姿态泄露”和“细节丢失”问题，具有重要的学术价值和工业应用前景。
